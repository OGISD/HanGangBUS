### Added (2026-07-08 한강 역류 감시)

  팔당댐 서식, 최신 강조 + 신선도). HRFCO 1019630 기간조회. buildCards가 행주대교 카드 바로 아래로 배치.
  ※ 역류 자동 판정(인천 조위 720cm=신곡수중보 초과)은 실측 검증 후 추가 예정.
  배경: 인천 조위>720cm면 밀물이 신곡수중보 넘어 상류 역류. 행주 절대수위 고정 임계값은 팔당댐 방류에
  따라 기준선이 변해 불안정 → 판정은 인천 720cm 기준으로 검증 후 도입 예정.

### Added
- 일출·일몰·월출·월몰: 인천 물때 카드 하단에 통합(별도 카드 아님). 한국천문연구원 출몰시각 API

- API 키 만료 사전 경고: 물때(2028-07-05)·출몰시각(2028-07-06) 키가 만료 60일 전부터 상단 배너로
  "미리 연장하세요" 알림(`API_EXPIRY`/`checkKeyExpiry`). 연장 시 배열의 date만 갱신.
- 물때 카드 화살표·증감 표시: 각 행에 만조 ▲/간조 ▼ + 직전 물때 대비 증감(cm, 내려가면 음수 예: -658).
  여의도를 만조 2개→하루 물때 4개 전부(+4시간)로 확대. 첫 행 증감은 어제 마지막 물때를 localStorage에 캐시해 계산.
- (대기) 팔당댐 주의선(2번째 절대값) — 회사 확인 후 `DAM.warnFlow`에 숫자만 넣으면 활성
- (대기) 일출·일몰·월출·월몰 — 한국천문연구원 출몰시각 API(RiseSetInfoService). **물때 카드 안에 통합**(별도
  카드 아님). 현재 키가 이 API에는 403(활성화 대기) — 승인·활성 후 구현.
- (연기) 센서 이상값 방어 1단계 — 기능 확장 뒤로 미룸

## [0.5.0] - 2026-07-06

### Added
- 인천 물때 카드: 국립해양조사원 조석예보(고저조) API로 인천(DT_0001)의 오늘 만조·간조 4개
  (시각·조위 cm) 표시. data.go.kr(1192136) 사용, XML 응답을 DOMParser로 파싱, CORS 허용이라 브라우저 직접 호출.
- 여의도 물때 카드: 인천 만조 시각 +4시간 = 여의도 만조 2개(자정 넘으면 "익일" 표기).
- 카드 배치: 팔당댐 → 잠수교 → 여의도물때 → 행주대교 → 인천물때 (buildCards가 교량 카드만
  재생성하고 물때 카드 사이에 삽입). 물때는 예보값이라 수동 모드에서도 표시.

## [0.4.0] - 2026-07-05

### Added
- 팔당댐 방류량 카드 (최상단): 현재 방류량 + 4시간 전 방류량(지금 여의도에 도달 중인 방류분).
  각 창은 10분 단위 3회 관측값을 **전부 개별 표시**(시각+값, 최신 행 강조), 두 열은 세로 구분선으로 분리,
  교량 카드와 동일한 데이터 신선도 경고. HRFCO 댐 API(팔당댐 1017310, tototf ㎥/s) 사용.
  수동 모드에서는 조회하지 않음(안내 문구 표시).
- 팔당댐 방류량 운항금지 경고: 각 열 최신 방류량이 3000㎥/s 이상이면 그 열에 빨간 경고
  ("여의도 운항금지" / "4시간 뒤 운항금지"). 주의선(2번째 절대값)은 회사 확인 후 추가 예정.
