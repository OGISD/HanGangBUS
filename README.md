# 한강 교량 통과높이 모니터

잠수교·행주대교의 선박 통과높이를 실시간 수위로 계산해 보여주는 단일 파일 모바일 웹앱.

## 빠른 시작
1. `index.html`을 열어 맨 위 `CONFIG.API_KEY`에 한강홍수통제소 인증키 입력
   (키는 hrfco.go.kr 오픈API 사용신청에서 무료 발급)
2. `index.html`을 정적 호스팅(Netlify Drop / GitHub Pages)에 올림
3. 생성된 링크를 공유 → 휴대폰 브라우저로 열기

키를 비워두면 수동 모드(수위 직접 입력)로 동작합니다. 키를 넣으면 자동 모드로 전환됩니다.

## 파일
- `index.html` — 웹앱 전체(단일 파일)
- `hrfco-proxy-worker.js` — (선택) Cloudflare Worker 프록시. 키를 숨기고 CORS 해결.
- `CLAUDE.md` — 프로젝트 맥락·공식·기준값 정리 (Claude Code가 이어서 작업할 때 참고)

## 공식
```
통과높이 = 기준높이 − 현재 수위
```
교량별 기준값과 근거는 `CLAUDE.md` 참고.

## 라이선스 / 주의
내부 공유용. 통과높이·임계값은 안전 관련 값이므로, 운영에 쓰기 전 회사 시스템 값과 대조하세요.
