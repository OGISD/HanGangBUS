# 알아두어야 할 특이사항

- **HRFCO API는 해외 IP 차단** — Cloudflare Worker·Vercel 등 무료 serverless 프록시 전부 불가
  (12초 타임아웃 후 522). 그래서 방문자 브라우저 직접 호출 구조이고, **API 키는 소스에 노출
  (의도된 결정)**. 어설픈 키 은닉 시도 금지 → [RULES.md](RULES.md) 금지 사항 참고.
- 해외에서 접속하면 자동 모드가 안 됨 → 수동 입력 폴백이 그 대비책.
- **물때 API(data.go.kr 1192136)는 `date` 파라미터를 무시하고 항상 서버 오늘(KST)만 반환.**
  필수 파라미터는 `obsCode`뿐. 응답은 XML(DOMParser 파싱). CORS 허용(*)이라 브라우저 직접 호출 OK.
  인천 `DT_0001`, 경인항 `DT_0058`(미사용). 키도 소스 노출(무료·읽기전용, HRFCO와 동일 판단).
- `hrfco-proxy-worker.js`는 폐기된 워커 잔재. 사용 안 함, 삭제해도 무방.
- 행주대교는 서해 조석 구간 → 썰물 때 수위 음수 정상. 급변 감지류 로직 넣을 때 오탐 주의.
- **사용자는 코딩 초보** — 멘토 모드(한 번에 한 단계, 완료 확인 후 진행) 필수. RULES.md 참고.
- 교량별 확정값·공식·검증 근거의 원본(source of truth)은 [CLAUDE.md](CLAUDE.md).
