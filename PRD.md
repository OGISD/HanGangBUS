## 프로젝트명

한강 교량 통과높이 모니터 (HanGangBUS)

---

## 한 줄 요약

한강 교량(잠수교·행주대교)의 **선박 통과높이**를 실시간 수위로 계산해 보여주는 단일 파일 모바일 웹앱.

---

## 배경 및 목적

- 한강에서 선박이 다리 밑을 지날 수 있는지는 "다리 밑 여유 높이(통과높이)"로 판단하는데,
  이 값은 수위에 따라 실시간으로 변한다.
- 회사 내부에서 필요할 때마다 수위를 찾아 직접 계산하는 대신, 휴대폰으로 링크만 열면
  현재 통과높이와 안전 판정(여유/위험/운항 금지)을 바로 볼 수 있게 한다.
- 공식: **통과높이 = 기준높이(base) − 현재 수위(wl)**, base = 형하 표고 − 수위계 영점표고(gdt).

---

## 대상 사용자

- 소수 내부 관계자(회사 동료 등). 링크 공유로 접근하며 일반 공개 홍보는 하지 않음.
- 모바일 브라우저 사용이 기본 시나리오.

---

## 핵심 기능 (Must Have)

1. 잠수교·행주대교의 통과높이를 실시간 수위(한강홍수통제소 오픈API)로 자동 계산·표시
2. 교량별 안전 판정: 운항 금지선(noGo)·위험 경고선(warn) 기준 색상/라벨 표시
3. 자동 호출 실패 시(해외 IP 등) 수동 수위 입력으로 계산 가능한 폴백
4. 데이터 신선도 경고: 관측값이 30분 이상 오래되면 빨간 ⚠ 표시
5. 참고용 면책 문구 상시 표시 (운항 판단은 공식 정보로 확인)

---

## 있으면 좋은 기능 (Nice to Have)

- 교량 추가(카드 확장) — `DEFAULT_BRIDGES` 배열에 항목만 추가하면 됨
- 센서 이상값 방어(수위 범위 sanity check)
- 행주대교 임계값을 실제 관심 선박(한강버스) air draft 기준으로 재설정

---

## 비범위 (Out of Scope)

- 회원/로그인, 서버 사이드 기능(DB 등) — 정적 단일 파일 유지
- 공식 항행 안전 시스템 대체 — 이 앱은 어디까지나 **참고용**
- API 키 완전 은닉 — HRFCO API가 해외 IP를 차단해 무료 serverless 프록시가 불가능하므로,
  키 노출을 감수하기로 결정함 (RULES.md 참고)

---

## 성공 기준 (Definition of Done)

- 링크를 열면 별도 조작 없이 두 교량의 현재 통과높이와 안전 판정이 보인다 ✅ (달성)
- 데이터가 오래되면 사용자가 알아챌 수 있다 ✅ (신선도 경고, 2026-06-22)
- 자동이 안 되는 환경에서도 수동 입력으로 쓸 수 있다 ✅ (달성)

---

## 제약 조건

- **무료로 유지**: 정적 호스팅(GitHub Pages) + 무료 오픈API만 사용
- **단일 파일**: index.html 하나에 HTML/CSS/JS 전부 포함 (배포·공유 단순화)
- **HRFCO API는 해외 IP 요청에 응답하지 않음** → 방문자 브라우저가 직접 호출(국내 IP)하는
  구조만 가능. Cloudflare Worker/Vercel 등 무료 serverless 프록시는 전부 불가.
- 안전 관련 수치(base, noGo, warn)는 회사 시스템 값과 대조·검증 후에만 반영
